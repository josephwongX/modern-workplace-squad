### Step 1

Select the Font Dialog Box Launcher located between the "Font" and "Paragraph".

![Step 1](step-1.png)

### Step 2

Select the font and size you desire then **Set As Default**.

![Step 2](step-2.png)

### Step 3

Select "_All documents based on the Normal template?_" then click **OK**!

![Step 3](step-3.png)
